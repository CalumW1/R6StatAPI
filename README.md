## Getting Started
you can install the package using npm or bun:

```bash
npm i r6statapi
```
or
```bash
bun add r6statapi
```
For a detailed guide on how to use the package, please refer to the [documentation](https://r6stats.esportsapp.gg/docs/getting-started/installation/)
